HR Analytics Dashboard (Power BI)
Overview

This project is an HR analytics dashboard built using Power BI.
It helps understand employee attrition, workforce structure, and job trends.

Files in this project
HR_Dashboard.pbix → Power BI dashboard file
HR_Report.pdf → Exported report
What this dashboard shows
Employee attrition analysis
Department-wise workforce breakdown
Job satisfaction insights
Salary and overtime patterns
Tools used
Power BI
DAX (basic calculations)
Power Query (data cleaning)
How to use
Download the PBIX file
Open it in Power BI Desktop
Explore the dashboard
